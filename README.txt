# Proyecto CIDT - API Inmobiliaria

## Descripción
Este proyecto implementa una API para una aplicación inmobiliaria (arrendamientos) 
utilizando FastAPI y SQLAlchemy.

## Requisitos
- Python 3.x
- [FastAPI](https://fastapi.tiangolo.com/)
- [SQLAlchemy](https://www.sqlalchemy.org/)
- Otros requisitos específicos que puedes encontrar en `requirements.txt`

## Configuración del Entorno Virtual
# Crear y activar un entorno virtual
python -m venv venv
source venv/bin/activate  # En Windows: venv\Scripts\activate

# Instalar dependencias
pip install -r requirements.txt
